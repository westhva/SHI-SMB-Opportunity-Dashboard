"We've been working with Canva to help consolidate that. From an IT perspective, the value is pretty straightforward:

  • SSO/SCIM integration so you're not managing another set of credentials
  • Centralized billing instead of 50 credit card subscriptions
  • Actually seeing who's using what through admin controls
  • Potential Adobe cost reduction - a lot of people have Creative Cloud seats when they really just need basic design capability"
